# HR-ANALYSIS-
 HR analytics project analyzing employee data to uncover insights on workforce trends, performance, and attrition. Includes data cleaning, KPI development, and interactive dashboard for data-driven decision-making.

## 🚀 Project Overview
This project delivers a comprehensive HR Analytics Dashboard designed to uncover workforce insights, track employee attrition, and support data-driven HR decisions.
Built using Power BI, the dashboard transforms raw HR data into clear, actionable intelligence across multiple dimensions such as employee demographics, salary distribution, job roles, and tenure.

## 🎯 Objectives
Analyze employee attrition patterns
Identify high-risk employee segments
Understand the impact of salary, tenure, and job roles on turnover
Enable HR teams to make strategic, data-backed decisions.

## 📌 Key Metrics
Total Employees: 1480
Average Tenure: 7.01 years
Average Salary: $6.5K
Attrition Count & Rate across multiple dimensions
